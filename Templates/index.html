About This Project

Providing analysis on United Airlines flight reliability for United hubs within the domestic US!

This project provides viewers with flight delay data, helping inform decision making when choosing what hubs to fly through.

In order to present this information, when the user opens the page, they are able to view a map with 2018 data from the Bureau of Transportation Transtats, analyzed to make compelling visualizations on the first map.

Meanwhile, scripts are running in the background to scrape the web and dynamically alter the page, displaying new content based on the scrape.

The second leaflet map displays the flight paths of United flights between their domestic hubs using Leaflet.Geodesic.

Additionally, viewers may view Plotly charts for statistics on historic United Airlines airport delays, cancellations, and on time departures for these hubs.
